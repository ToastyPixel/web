The Silent Mine is a just-for-fun sandbox game experiment — basically a Minecraft-ish block world project I mess with when I’m in the mood.

It’s still in beta and definitely has bugs. Updates are pretty random because this is more of a learning/playground project than something I’m seriously shipping.

Platform note: right now it’s only been tested on Linux, so that’s the only OS I can say it works on. I haven’t tested it on Windows or macOS yet.

If there’s anything new, it’ll be on the releases page.
